Welcome California Accident Stats Explorer

**Description:**
Our project developed a website to present and analyze traffic accident data in California, using statistical methods to identify accident patterns from 2019 to 2021 and emphasizing causes ranging from alcohol influence to road conditions. The site aims to boost safety awareness with interactive tools and visuals, helping users understand and avoid hazardous driving situations.

**Instructions for using: **
Please npm install in our server file than npm start in server file.
Than npm install in our client file.
Please (npm install chart.js react-chartjs-2) and (npm install axios)in client file
Than npm start in client.
